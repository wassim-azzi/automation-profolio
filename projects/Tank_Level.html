Automatic Tank Level Control System

Industrial Automation Project

Industrial automation project developed using Siemens STEP 7 (LAD) and HMI simulated in Factory I/O

Project Overview

The Automatic Tank Level Control System is an industrial automation project developed using GRAFCET, Siemens STEP 7 (LAD), Factory I/O, and an HMI. The system automatically controls the filling and draining of a tank while providing both Automatic and Manual operating modes through the HMI for flexible process control.

Operating Sequence

  1. Select Automatic Mode on the HMI.
  2. Enter the desired tank level (e.g., 60%, 80%, or 100%).
  3. The PLC compares the target level with the current tank level.
  4. If the current level is below the target, the filling valve/pump is activated.
  5. If the current level is above the target, the draining valve/pump is activated.
  6. Once the target level is reached, the PLC stops the process automatically.
